# BlossomCalendar

BlossomCalendar is a lightweight macOS menu bar application built with SwiftUI. it provides a quick-access calendar with local event management and customizable blossom-inspired themes, allowing you to manage your schedule directly from your menu bar.

## Getting Started

### Installation
Clone the repository to your local machine:
```bash
git clone https://github.com/Paul-CodeGH/BlossomCalendar_macOS.git
```

### Opening in Xcode
1. Navigate to the cloned directory: `cd BlossomCalendar_macOS`
2. Open the project file: `open BlossomCalendar.xcodeproj`
3. In Xcode, select the **BlossomCalendar** target and press **Cmd + R** to build and run the application.
